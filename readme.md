# ☁️ Cloud Storage Web Application

Веб-приложение облачного хранилища, разработанное в рамках дипломного проекта.
Позволяет пользователям загружать, хранить, скачивать и управлять файлами через удобный веб-интерфейс.

---

## 🚀 Основные возможности

### 👤 Пользователи

* Регистрация с валидацией данных
* Аутентификация (login/logout)
* Ролевая модель (обычный пользователь / администратор)

### 🗂️ Файловое хранилище

* Загрузка файлов
* Скачивание файлов
* Удаление файлов
* Переименование
* Добавление комментариев
* Генерация публичной ссылки
* Скачивание по публичной ссылке

### 🛠️ Администрирование

* Просмотр списка пользователей
* Назначение / снятие прав администратора
* Доступ к файлам всех пользователей
* Фильтрация файлов по пользователю

## ⚙️ Установка и запуск

### 🔧 Требования

* Docker
* Docker Compose
* Nginx

---

### ▶️ Запуск проекта

```bash
git clone https://github.com/Valdemarovna/cloud-diploma
cd cloud-diploma/cloud_storage_backend

docker compose up --build
```
